About This Project

This website was created in compliance to the developers final project in COSC 203 (Design and Analysis Algorithm) where they were tasked to solve a real-world problem using basic algorithms.

In recent discussions online, Goodreads List has been a topic of major concern, particularly with its lack of a proper search function. Readers and authors that use this feature are forced to scroll all the way down and click multiple "next" buttons to seek for a particular book. With this in mind, the developers concocted a combination of Quick Sort and Binary Search algorithms to effectively search for a book inside said lists. To test the efficiency of the algorithms in the real-world, the simulation website has produced three lists that contains 20, 100, and 1001 books respectively. This setup ensures a fair and practical evaluation of the search and sort functions in handling lists of different sizes.
